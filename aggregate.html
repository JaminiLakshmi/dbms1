AGGREGATE

  • An SQL aggregate function calculates on a set of values and returns a single value
  • sum()-sum(col_name):Sum all Non Null values of Column col_name $$$$$$$$ sum(Distinct col_name): Sum of all distinct Non-Null values
  • average() or avg()-Avg(col_name) = Sum(col_name) / count(col_name) $$$$$$$$Avg(Distinct col_name) = sum(Distinct col_name) / Count(Distinct col_name)
  • max()-Max(col_name): Maximum value in the col_name
  • min()-Min(col_name): Minimum value in the col_name column except NULL
  • count()-Count(*): Returns total number of records $$$$$$$$$$ Count(col_name): Return number of Non Null values over the column $$$$$$$$$$$ Count(Distinct col_name): Return number of distinct Non Null values over the column
