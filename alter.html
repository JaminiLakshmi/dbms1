ALTER

  • The ALTER TABLE statement is used to add, delete, or modify columns in an existing table. The ALTER TABLE statement is also used to add and drop various constraints on an existing table.
  • ADD-add is used to add column to the existing table
  • modify-modify is used to change the datatype of the existing column
  • change-change is used to change the colume name to the new column name
  • drop-drop is used to delete the exisiting column
  • rename-rename is used to change the existing table name to the new name
